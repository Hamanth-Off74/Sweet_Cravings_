========================================
HOW IT WORKS
========================================

1. Customer places order with Razorpay payment
2. Payment is verified
3. Order saved to MongoDB with:
   - Unique Order ID
   - Customer details
   - Delivery address
   - Items ordered
   - Payment information
   - Razorpay transaction IDs
   - Status: pending
   
4. Admin can view all orders at /admin
5. See customer details and order information
6. Track order status

========================================
MONGODB COMMANDS (Optional)
========================================

To view data in MongoDB Shell:

1. Open MongoDB Shell:
   mongosh

2. Switch to database:
   use sweetcravings

3. View all orders:
   db.orders.find().pretty()

4. Count orders:
   db.orders.countDocuments()

5. Find specific order:
   db.orders.find({orderId: "ORD-123456"})

========================================
TROUBLESHOOTING
========================================

Problem: "MongoDB Connection Error"
Solution: Make sure MongoDB is running
         Windows: Check MongoDB service in Services
         Mac: brew services start mongodb-community

Problem: Can't access /admin page
Solution: Server must be running (node app.js)
         Visit: http://localhost:5000/admin

Problem: Orders not showing
Solution: Place a test order first
         Orders appear after payment completion
