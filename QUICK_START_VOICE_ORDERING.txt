┌─────────────────────────────────────────────────────────────────┐
│ STEP 1: GET API KEY                                             │
└─────────────────────────────────────────────────────────────────┘
  1. Visit: https://www.assemblyai.com/
  2. Sign up (free - no credit card needed)
  3. Copy your API key from dashboard

┌─────────────────────────────────────────────────────────────────┐
│ STEP 2: CONFIGURE                                               │
└─────────────────────────────────────────────────────────────────┘
  Windows:
    .\setup-voice-ordering.ps1
  
  Or manually:
    1. Copy .env.example to .env
    2. Open .env in notepad
    3. Replace: ASSEMBLYAI_KEY=your_actual_api_key_here
    4. Save file

┌─────────────────────────────────────────────────────────────────┐
│ STEP 3: START SERVERS                                           │
└─────────────────────────────────────────────────────────────────┘
  Backend (if not running):
    npm start
  
  Frontend (if not running):
    cd client
    npm run dev

┌─────────────────────────────────────────────────────────────────┐
│ STEP 4: TEST IT OUT                                             │
└─────────────────────────────────────────────────────────────────┘
  1. Open browser: http://localhost:3000/menu
  2. Find purple "Voice Order" section
  3. Click "Start Recording"
  4. Say: "I want two chocolate cakes and one vanilla ice cream"
  5. Click "Stop & Process"
  6. Check your cart!

╔═══════════════════════════════════════════════════════════════════╗
║                         TROUBLESHOOTING                           ║
╚═══════════════════════════════════════════════════════════════════╝

  Problem: Microphone doesn't work
  → Check browser permissions (Settings > Privacy)
  → Try refreshing the page
  
  Problem: "ASSEMBLYAI_KEY not configured"
  → Make sure .env file exists in root folder
  → Check API key is correct
  → Restart backend server
  
  Problem: No items detected
  → Speak more clearly
  → Use full dessert names
  → Check dessert is in the menu
  
  Problem: Server not starting
  → Check MongoDB is running
  → Verify all dependencies installed: npm install
