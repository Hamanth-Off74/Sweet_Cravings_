⚠️ CRITICAL: BEFORE TESTING
========================================

You MUST get your Razorpay API keys first!

1. Sign up at: https://razorpay.com/
2. Go to Settings → API Keys
3. Copy your TEST keys (starts with "rzp_test_")

4. Open: app.js
5. Replace these lines:
   
   key_id: 'rzp_test_YOUR_KEY_ID',      ← Paste your Key ID here
   key_secret: 'YOUR_KEY_SECRET'         ← Paste your Key Secret here

6. Save and restart server:
   node app.js
