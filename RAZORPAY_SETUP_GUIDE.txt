========================================
STEP 1: GET RAZORPAY API KEYS
========================================

1. Go to https://razorpay.com/
2. Click "Sign Up" to create a new account
3. Complete the registration process
4. Once logged in, go to Settings → API Keys (https://dashboard.razorpay.com/app/keys)
5. You'll see TWO sets of keys:
   - Test Keys (for testing payments)
   - Live Keys (for real payments)

6. For TESTING, use Test Keys:
   - Key ID: starts with "rzp_test_"
   - Key Secret: a long alphanumeric string

7. For PRODUCTION, use Live Keys (after your business is verified):
   - Key ID: starts with "rzp_live_"
   - Key Secret: a long alphanumeric string

========================================

========================================

1. Open: c:\Users\Sun Tyres\OneDrive\Desktop\dessert web\app.js

2. Find these lines (around line 8-9):

   key_secret: 'YOUR_KEY_SECRET'

========================================
WHAT'S BEEN IMPLEMENTED
========================================

✅ Backend Integration (app.js):
   - Razorpay SDK installed and configured
   - Razorpay instance created with API keys

✅ Payment Routes (routes/index.js):
   - POST /create-razorpay-order
     → Creates Razorpay order
     → Converts amount to paise (₹ × 100)
     → Returns order_id and key_id
   
   - POST /verify-payment
     → Verifies payment signature

   - Error handling and notifications

✅ Checkout Page (views/checkout.ejs):
   - Razorpay checkout.js script included
   - Payment method selection updated
   - Razorpay and Cash on Delivery options

========================================
PAYMENT FLOW OVERVIEW
========================================

1. User fills checkout form
2. Clicks "Proceed to Payment"
3. Frontend calls /create-razorpay-order
4. Backend creates Razorpay order
5. Frontend opens Razorpay modal with order_id
6. User enters card details and pays
7. Razorpay processes payment
8. On success, frontend calls /verify-payment
9. Backend verifies payment signature
10. Order saved with payment details
11. Cart cleared, success modal shown

========================================
TROUBLESHOOTING
========================================

Problem: Payment modal doesn't open
Solution: Check if Razorpay script is loaded in checkout.ejs
         Check browser console for errors

Problem: Payment fails every time
Solution: Verify API keys are correct
         Make sure you're using TEST keys for testing
         Check if amount is being sent correctly

Problem: "Invalid key_id" error
Solution: You haven't replaced the placeholder keys
         Go to app.js and add your actual Razorpay keys

Problem: Payment succeeds but order not saved
Solution: Check /verify-payment route logs
         Ensure orderManager is initialized
