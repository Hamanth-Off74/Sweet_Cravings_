  • Responsive CSS

Backend

  • Node.js + Express
  • AssemblyAI SDK

Key Features

🎤 Voice Recording

Browser-based audio recording with MediaRecorder API. Works on all modern browsers with microphone permission.

🗣️ Natural Language

Users can speak naturally: "I want two chocolate cakes and one vanilla ice cream" - no rigid commands needed.

🤖 AI-Powered

AssemblyAI's advanced speech recognition provides accurate transcription with confidence scores.

🔍 Smart Parsing

Intelligent algorithm matches dessert names using fuzzy logic and extracts quantities from text.

🛒 Auto Cart

Detected items automatically added to cart with correct quantities - no manual clicking needed.

✨ Beautiful UI

Gradient purple design with animated indicators, loading states, and clear feedback messages.
